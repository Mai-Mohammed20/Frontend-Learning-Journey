HTML
HTML (HyperText Markup Language) is the main language used to create the structure of web pages.
It organizes the content on a page by defining elements such as headings, paragraphs, images, links, tables, and lists.
HTML tells the browser what each part of the content is and how it should be arranged on the page. It works as the skeleton of a web page,
providing the basic layout, while CSS is used for design and styling, and JavaScript is used to add interactivity and dynamic behavior.
CSS
CSS (Cascading Style Sheets) is used for styling web pages.
It controls the appearance of HTML elements, including colors, fonts, sizes, spacing, layout, and positioning.
CSS makes web pages visually attractive and consistent, separating the design from the content.
While HTML provides the structure of a web page, CSS defines how that structure looks on the screen.
